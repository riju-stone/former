# Former
An elegant form builder platform, inspired from Google forms & Typeform

## Usage & Features

- Elegant & intuitive editor UI for granular form building experience
- Ability to create multi-step forms
- Supports multiple form element types - Text, Long Text, Options, Date and so on
- Supports validation constraints for each form element
